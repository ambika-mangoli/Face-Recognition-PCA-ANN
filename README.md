# Face Recognition using PCA and ANN

##  Project Description
This project implements a Face Recognition System using:
- PCA (Principal Component Analysis)
- ANN (Artificial Neural Network)

The system recognizes faces from a dataset and also detects unknown (imposter) faces using Euclidean distance thresholding.

---

##  Features
- Face Recognition
- Eigenface Feature Extraction
- ANN Classification
- Imposter Detection
- Accuracy vs k Analysis

---

##  Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn
- Matplotlib

---

## How to Run

### Train Model
python train.py

### Test Model
python test.py

## Results

- Best k value: 20
